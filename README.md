# \# STM32 DHT11 + OLED 温湿度采集项目

# 基于 STM32F103C8T6 的温湿度采集系统，使用 DHT11 传感器 + 0.96寸 I2C OLED 显示屏，支持串口打印。

# 

# \## 功能

# \- DHT11 温湿度读取

# \- OLED 中文显示

# \- 串口定时输出数据

# \- 代码模块化，易移植

# 

# \## 硬件接线

# \- OLED SCL -> PB6

# \- OLED SDA -> PB7

# \- DHT11 DATA -> PA6

# \- 全部 3.3V 供电

# 

# \## 软件环境

# \- Keil MDK5

# \- STM32CubeMX

# \- GitHub Desktop

# 

# \## 运行效果

# \- OLED 显示：温湿度采集、温度、湿度

# \- 串口每2秒输出一次温湿度数据

# \- 稳定不卡顿、不报错
