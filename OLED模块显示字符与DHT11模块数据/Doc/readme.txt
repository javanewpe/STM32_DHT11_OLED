						【DHT11 温湿度采集 + OLED 显示 项目说明文档】
*********************************************************************************************
【项目简介】
本项目基于 STM32 平台，实现 DHT11 温湿度传感器数据读取，
并通过 I2C OLED（0.96 寸 SSD1306） 实时显示，同时支持串口打印数据。

【硬件平台】
STM32F103C8T6（最小系统板）
外设模块
DHT11 温湿度传感器
0.96 寸 I2C OLED（白色屏，I2C 地址：0x3C）
USB-TTL 串口模块（用于查看打印信息）

【引脚分配】
NRST        -> NRST           		// 核心板复位引脚，板载已上拉并带硬件消抖  
DEBUG_TX    -> USART1_TX -> PA9  	// 串口1发送，用于调试打印输出（可选）  
DEBUG_RX    -> USART1_RX -> PA10  	// 串口1接收，用于调试（可选）  

IIC_SCL     -> PB6           		// OLED I2C通信的时钟线（SCL）  
IIC_SDA     -> PB7           		// OLED I2C通信的数据线（SDA）

DHT11 VCC → 3.3V
DHT11 GND → GND
DHT11_DATA  → PA6	  				// DHT11单总线数据引脚

OLED VCC → 3.3V
OLED GND → GND
OLED SCL → PB6
OLED SDA → PB7

【程序描述】
1. 初始化串口（可选，用于调试打印）  
2. 初始化I2C接口，用于与OLED屏幕通信  
3. 初始化OLED，清屏并显示"野火电子"字符  
4. 初始化DHT11 GPIO端口  
5. 主循环中周期性读取DHT11数据并刷新OLED显示内容 

【功能特点】
1. 稳定读取 DHT11 温湿度（整数 + 小数）
2. OLED 显示中文 + 数字 + 符号
3. 串口定时打印温湿度数据
4. 代码分层清晰、易移植、易维护
5. 延时使用 DWT 精准定时器，不占用硬件定时器

【注意事项】
1. OLED 绝对不能接 5V，会烧坏屏幕
2. DHT11 不支持高速读取，必须 ≥ 2 秒读取一次
3. I2C 引脚必须开启复用功能
4. 代码已做防卡死处理，适合长期运行
